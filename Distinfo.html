3.2 Invoking the command line tool

This is the output of imogene distinfo --help: 

     Imogene 0.1
     
     Distance between PWMs.
     
     Usage: imogene [OPTIONS]... [<motifs>]...
     
       -h, --help              Print help and exit
       -V, --version           Print version and exit
       -w, --width=int         Width of the motifs  (default=`10')
       -t, --threshold=double  Threshold used for motif scanning
       -s, --species=str       Species used for motifs generation.
       -m, --motifs=str        file containing a list of motif definitons

You mainly need to provide two PWM files in the form of: 

     enhancer1 2L 2345  3456
     enhancer2 2R 4567  5678

You can find example files in the folder example of the tarball. You can then try: 

     imogene distinfo --input example/PWM1.dat example/PWM2.dat
