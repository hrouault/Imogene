6 Running test files

6.1 Extraction

For drosophila, there is a file examples/extract/coords-droso.dat that can be used as an input for imogene extract: 

     imogene extract -s droso --input examples/extract/coords-droso.dat

It creates a folder align containing the extracted fasta formatted alignments.

6.2 Running genmot on the extracted sequences

You now have to run genmot on the extracted alignments 

     imogene genmot -s droso -a align
