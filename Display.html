5 Displays motifs on sequences

5.1 Invoking the command line tool

This is the output of imogene display --help: 

     Imogene beta.1-101-3001-dirty
     
     Display motifs instances on sequences, with conservation shown in italics.
     
     Usage: imogene [OPTIONS]... [<motifs>]...
     
       -h, --help              Print help and exit
       -V, --version           Print version and exit
       -t, --threshold=double  Threshold used for motif scanning  (default=`9')
       -s, --species=str       Species used for motifs generation.
       -m, --motifs=str        file containing a list of motif definitions
       -n, --nbmots=int        Number of motifs to display (max=4)  (default=`4')
       -a, --align=str         Folder containing the fasta formatted (.fa) files of
                                 enhancer alignments.
           --tex-ref           Output is tex formatted. Display only reference
                                 species sequence.
           --html-ref          Output is html formatted. Display only reference
                                 species sequence.
           --tex-align         Output is tex formatted. Display the entire
                                 alignment.
           --svg               Output is svg formatted.

You need to provide a motif file as generated by the genmot mode, and the path to the folder containing the fasta (.fa) formatted alignments you wish to scan. These alignments can be automatically generated from a coordinate file using the extract mode. Depending on the chosen mode, the output will be the tex formatted sequences (tex-ref and tex-align), or a svg formatted graphical representation of motifs instances on the sequences (svg). You can find example files in the folder example of the tarball. You can then try: 

     imogene display -s eutherian -m motifs.txt -a align --tex-ref
     imogene display -s eutherian -m motifs.txt -a align --tex-align
     imogene display -s eutherian -m motifs.txt -a align --svg

5.2 Generating motif logo

This mode uses the python modules corebio and weblogo. You can install them with: 

     > pip install corebio
     > pip install weblogo
